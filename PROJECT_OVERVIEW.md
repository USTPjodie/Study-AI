# Project Overview

## Objectives

StudyAI aims to provide Computer Engineering students with an intelligent, accessible study companion that can:

1. Explain complex technical concepts in simple terms
2. Generate practice quizzes for self-assessment
3. Summarize lecture notes and textbooks into study-friendly formats
4. Maintain a persistent history of study sessions

## Scope

### In Scope
- User authentication with email verification
- Real-time AI chat powered by Groq API
- Three distinct AI modes: Chat, Quiz, and Note Summarizer
- Session persistence using Supabase PostgreSQL
- Responsive web interface (desktop + mobile)
- Dark/light theme support
- Chat export functionality

### Out of Scope
- Offline support
- Native mobile apps
- Real-time collaborative study rooms
- Integration with learning management systems (LMS)

## Target Users

- **Primary**: Computer Engineering undergraduate students (1st–4th year)
- **Secondary**: Self-taught programmers and bootcamp students
- **Use cases**:
  - Preparing for exams on operating systems, networks, data structures
  - Reviewing and summarizing lecture notes
  - Self-testing with AI-generated quizzes
  - Getting quick explanations for confusing topics

## Key User Stories

1. As a student, I want to ask questions about CPU scheduling so I can understand it before my OS exam.
2. As a student, I want to generate quizzes on data structures so I can test my knowledge.
3. As a student, I want to paste my notes and get a summary so I can study more efficiently.
4. As a student, I want my chat history saved so I can review past explanations.
